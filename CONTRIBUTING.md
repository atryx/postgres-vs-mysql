# Contributing

Thanks for helping improve this comparison! Here's how to contribute.

## Guidelines

- **Stay neutral** — this is a factual comparison, not an advocacy page
- **Cite sources** — link to official docs, benchmarks, or reputable articles
- **Keep it current** — update version numbers and dates when information changes
- **Use tables** — consistent with the existing format

## What to contribute

- Corrections to inaccurate information
- Updated benchmarks or pricing
- New comparison categories
- Better explanations or examples
- Typo and formatting fixes

## How to submit

1. Fork this repository
2. Create a branch (`git checkout -b fix/update-pricing`)
3. Make your changes
4. Submit a pull request with a clear description of what changed and why

## Code of conduct

Be respectful and constructive. We're all here to help developers make better decisions.
